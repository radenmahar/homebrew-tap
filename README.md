# radenmahar/homebrew-tap

Personal Homebrew tap for my own CLI tools.

## Usage

```bash
brew tap radenmahar/tap
brew install hopdrop
```

## Formulae

| Formula | Description |
|---|---|
| [`hopdrop`](Formula/hopdrop.rb) | Send files from your Mac to your Android phone over WiFi, one command. [Source](https://github.com/radenmahar/hopdrop) |
